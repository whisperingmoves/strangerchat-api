# 陌聊 App

跨平台陌生人聊天 App 服务端接口服务。

[客户端代码](https://github.com/whisperingmoves/strangerchat-app)

## 修改配置文件

将 .env.sample 文件拷贝成一个新文件，并将文件重命名为 .env ，然后修改 .env 文件

## 运行应用程序栈

```bash
docker compose up -d
```

## 查看日志

```bash
docker compose logs -f
```
